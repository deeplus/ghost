# 5.5 变换

# Transform


Transform 可以在其子组件绘制时对其应用一些矩阵变换来实现一些特效。Matrix4 是一个4D矩阵，通过它我们可以实现各种矩阵操作。


    1. @required Matrix4 -- transform，
    1. Offset -- origin，坐标系原点
    1. AlignmentGeometry -- alignment，对齐方式
    1. bool -- transformHitTests: true，
    1. Widget -- child，

Matrix4 相关方法：

    1. rotationX(double -- radians)、rotationY(double -- radians)、rotationZ(double -- radians)，沿X、Y、Z轴旋转。
    1. skew(double -- alpha, double -- beta)，绕X、Y轴倾斜。
    1. skewX(double -- alpha)、skewY(double -- beta)，同上。

Transform部件综合示例：

Container(
    color: Colors.black,
    child: Transform(
        alignment: Alignment.topLeft, //相对于坐标系原点对齐
        transform: Matrix4.skewY(0.2), //沿Y轴倾斜0.3弧度
        child: Container(
            padding: EdgeInsets.all(8),
            color: Colors.deepOrange,
            child: Text('Apartment for rent!'),
        ),
    ),
),

# 平移


Transform.translate 接收一个offset参数，可以在绘制时沿x、y轴对子组件平移指定的距离。


    1. @required Offset -- offset，
    1. bool -- transformHitTests: true，
    1. Widget -- child，

translate示例：

DecoratedBox(
    decoration:BoxDecoration(color: Colors.red),
    //默认原点为左上角，向左移20像素，向上平移5像素
    child: Transform.translate(
        offset: Offset(-20, -5),
        child: Text("Hello world"),
    ),
),

# 旋转


Transform.rotate 可以对子组件进行旋转变换。


    1. @required double -- angle，旋转角度，单位为弧度。
    1. Offset -- origin，
    1. AlignmentGeometry -- alignment: Alignment.center
    1. bool -- transformHitTests: true
    1. Widget -- child

rotate示例：

import 'dart:math' as math;

DecoratedBox(
    decoration:BoxDecoration(color: Colors.red),
    child: Transform.rotate(
        //旋转90度
        angle: math.pi / 2,
        child: Text("Hello world"),
    ),
),

# 缩放


Transform.scale 可以对子组件进行缩小或放大。


    1. @required double -- scale，缩放比例
    1. Offset -- origin
    1. AlignmentGeometry -- alignment: Alignment.center,
    1. bool -- transformHitTests: true
    1. Widget -- child

scale示例：

DecoratedBox(
    decoration:BoxDecoration(color: Colors.red),
    child: Transform.scale(
        scale: 1.5, //放大到1.5倍
        child: Text("Hello world")
    )
),
