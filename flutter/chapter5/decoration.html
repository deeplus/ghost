# 5.3 装饰类容器

# DecoratedBox


DecoratedBox可以在其子组件绘制前(或后)绘制一些装饰（Decoration），如背景、边框、渐变等。


    1. @required Decoration -- decoration，
    1. DecorationPosition -- position: DecorationPosition.background，
    1. Widget -- child，

BoxDecoration相关属性：

    1. Color -- color，背景颜色
    1. DecorationImage -- image，背景图片
    1. BoxBorder -- border，边框
    1. BorderRadiusGeometry -- borderRadius，圆角
    1. List<BoxShadow> -- boxShadow，阴影，可以指定多个
    1. Gradient -- gradient，渐变
    1. BlendMode -- backgroundBlendMode，背景混合模式
    1. BoxShape -- shape: BoxShape.rectangle，形状

DecoratedBox部件示例：

DecoratedBox(
	decoration: BoxDecoration(
        gradient: LinearGradient(
            colors: [Colors.red,Colors.blue], // 背景渐变
        ),
        borderRadius: BorderRadius.circular(5), // 5像素圆角
        boxShadow: [ // 阴影
            BoxShadow(
                color: Colors.black54,
                offset: Offset(2, 2),
                blurRadius: 4,
            )
        ]
	),
	child:Padding(
        padding: EdgeInsets.symmetric( // 对称，水平垂直内边距
            horizontal: 80.0,
            vertical: 18.0
        ),
        child: Text('Login',
			style: TextStyle(
				color: Colors.white
			),
		),
	),
),
