# 3.1 文字、文字样式

# Text


Text 用于显示简单样式文本，它包含一些控制文本显示样式的一些属性。


    1. TextAlign -- textAlign, 多行文本对齐方式 （center居中，left左对齐，right右对齐，justfy两端对齐）
    1. TextDirection -- textDirection, 文本方向（ltr从左至右，rtl从右至左）
    1. int -- maxLines, 文字显示最大行数
    1. bool -- softWrap, 是否自动换行（true自动换行，false单行显示，超出屏幕部分默认截断处理）
    1. TextOverflow -- overflow, 文字超出屏幕之后的处理方式（clip裁剪，fade渐隐，ellipsis省略号）
    1. double -- textScaleFactor, 字体显示倍率
    1. TextStyle -- style, 字体样式

# TextStyle


TextStyle 用于指定文本显示的样式如颜色、字体、粗细、背景等。


    1. Color -- color, 字体的颜色
    1. double -- fontSize, 文字大小,单位为像素，如果没有指定大小则默认为14像素，可以乘以textScaleFactor来增加字体大小以便用户更加方便的阅读
    1. FontWeight -- fontWeight, 字体厚度，可以使文本变粗或变细
    1. FontStyle -- fontStyle, 字体变形，有两种 FontStyle.normal（字体直立）， FontStyle.italic（字体倾斜）
    1. double -- letterSpacing, 字母间距，整数拉开字母距离，若是负数则拉近字母距离
    1. double -- wordSpacing, 单词间距，同上
    1. TextBaseline -- textBaseline, 用于对齐文本的水平线
    1. double -- height, 文本行高，为字体大小的倍数
    1. Locale -- locale, 用于选择区域特定符号的区域设置
    1. Paint -- foreground,
    1. Color -- backgroundColor, 文本的背景颜色
    1. String -- fontFamily,
    1. List<Shadow> -- shadows, 文本的阴影可以利用列表叠加处理，例如shadows: [Shadow(color:Colors.black,offset: Offset(6, 3), blurRadius: 10)], color即阴影的颜色， offset即阴影相对文本的偏移坐标，blurRadius即阴影的模糊程度，越小越清晰
    1. TextDecoration -- decoration, 文字的线性装饰，比如 underline 下划线， lineThrough删除线
    1. Color -- decorationColor, 文本装饰线的颜色
    1. TextDecorationStyle -- decorationStyle, 文本装饰线的样式，比如 dashed 虚线
    1. String -- debugLabel, 这种文本样式的可读描述，此属性仅在调试构建中维护。
    1. bool -- inherit: true, 是否将null值替换为祖先文本样式中的值（例如，在TextSpan树中。如果为false，则没有显式值的属性将恢复为默认值：白色，字体大小为10像素，采用无衬线字体。）

文本和文本样式综合示例：

Text(
    'hello world',
    style: TextStyle(
        fontSize: 24,
        height: 2,
        fontWeight: FontWeight.w700,
        fontStyle: FontStyle.italic,
        color: Color(0xaaff0000),
        letterSpacing: 1.2,
        wordSpacing: 2.5,
        shadows: [
            Shadow(
                color: Colors.purple,
                offset: Offset(2, 2)
            )
        ],
        decoration: TextDecoration.lineThrough,
        decorationColor: Colors.blue
    ),
)

# TextSpan


有时候我们需要对一个Text内容的不同部分按照不同的样式显示，这时就可以使用 TextSpan，它代表文本的一个“片段”。


# Text.rich和RichText


Text有一个静态方法rich，使用 Text.rich 可以达到 RichText 的效果。


TextSpan和RichText综合示例：

Text.rich(
    TextSpan(
    	text: '登陆即同意',
        style: TextStyle(fontSize: 14, color: Colors.black),
        children: [
            TextSpan(
                text: '"服务条款"',
                style: TextStyle(fontSize: 14, color: Colors.blue),
                recognizer: TapGestureRecognizer()
                ..onTap = () {
                    print('点击了解服务条款');
                }
            ),
            TextSpan(
                text: '和',
                style: TextStyle(fontSize: 14, color: Colors.black),
            ),
            TextSpan(
                text: '"隐私政策"',
                style: TextStyle(fontSize: 14, color: Colors.blue),
                recognizer: TapGestureRecognizer()
                ..onTap = () {
                    print('点击了解隐私政策');
                }
            )
        ]
    )
)
