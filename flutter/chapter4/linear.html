# 4.2 线性布局

  • 所谓线性布局，即指沿水平或垂直方向排布子部件。

  • Flutter 中通过Row和Column来实现线性布局，Row和 Column都继承自Flex 。

  • Row 和 Column 都只会在主轴方向占用尽可能大的空间，而纵轴的宽度则取决于他们最大子元素的长度。


# Row

Row 的主轴是水平方向，可以在水平方向排列其子widget。


    1. TextDirection -- textDirection，表示子组件在主轴方向的排列顺序（是从左往右还是从右往左）；
    1. MainAxisSize -- mainAxisSize: MainAxisSize.max，表示 Row本身 在主轴方向所占用的空间；
      • MainAxisSize.max，表示尽可能多的占用水平方向的空间，此时无论子widgets实际占用多少水平空间，Row的宽度始终等于水平方向的最大宽度；
      • MainAxisSize.min表示尽可能少的占用水平空间，当子组件没有占满水平剩余空间，则Row的实际宽度等于所有子组件占用的的水平空间；
    1. MainAxisAlignment -- mainAxisAlignment: MainAxisAlignment.start，表示子组件在Row所占用的水平空间内对齐方式；
      • 如果 mainAxisSize 值为 MainAxisSize.min ，则此属性无意义，因为子组件的宽度等于 Row的 宽度；
      • 只有当 mainAxisSize 的值为 MainAxisSize.max 时，此属性才有意义；
    1. CrossAxisAlignment -- crossAxisAlignment: CrossAxisAlignment.center，表示子组件在纵轴方向的对齐方式；
      • Row的高度等于子组件中最高的子元素高度，它的取值和MainAxisAlignment一样(包含start、end、 center三个值)，不同的是crossAxisAlignment的参考系是verticalDirection；
      • verticalDirection 值为 VerticalDirection.down 时 crossAxisAlignment.start 指顶部对齐
      • verticalDirection 值为 VerticalDirection.up 时，crossAxisAlignment.start 指底部对齐；
      • crossAxisAlignment.end 和 crossAxisAlignment.start 正好相反；
    1. VerticalDirection -- verticalDirection: VerticalDirection.down，表示Row纵轴（垂直）的对齐方向，默认是VerticalDirection.down，表示从上到下；
    1. TextBaseline textBaseline，设置文本基线的基准值；
    1. List<Widget> children: const [] ，子部件数组。

Row部件示例：

Container(
    color: Colors.red,
    alignment: Alignment.center,
    child: Row(
        mainAxisAlignment: MainAxisAlignment.spaceBetween,
        crossAxisAlignment: CrossAxisAlignment.start,
        textDirection: TextDirection.rtl,
        verticalDirection: VerticalDirection.up,
        children: <Widget>[
            Container(
                width: 100,
                height: 100,
                color: Colors.deepPurple,
                child: Text('a'),
            ),
            Container(
                width: 100,
                height: 200,
                color: Colors.blue,
                child: Text('b'),
            ),
            Container(
                width: 120,
                height: 120,
                color: Colors.white,
                child: Text('c'),
            ),
        ],
    ),
),

# Column

可以在垂直方向排列其子widget，参数和 Row 一样，不同的是主轴方向是垂直方向。


Column 部件示例：

Container(
    color: Colors.red,
    alignment: Alignment.topLeft,
    child: Column(
        mainAxisAlignment: MainAxisAlignment.spaceEvenly,
        crossAxisAlignment: CrossAxisAlignment.end,
        children: <Widget>[
            Container(
                width: 100,
                height: 100,
                color: Colors.deepPurple,
                child: Text('a'),
            ),
            Container(
                width: 200,
                height: 100,
                color: Colors.blue,
                child: Text('b'),
            ),
            Container(
                width: 120,
                height: 100,
                color: Colors.white,
                child: Text('c'),
            ),
        ],
    ),
),
